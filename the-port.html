The Port Fish & Chips

The Port is a fish & chips restaurant located on Manitoulin Island in Northern Ontario. The business partners Steph and Jared were looking to extend their pre-existing branding into a website that could serve as a platform for announcements. The site would also function as a source of information and marketing that could help the Port tap into the summer cottage market.

Working with developer Sav Nardi, we designed and began developing a responsive website. The website is currently in development.

Extending the Brand

The existing icons designed by Kendra Edwards were extended into a set that included menu, open/closed, and weather themed icons.

A weather icon would be placed in the navigation bar for the website: the specific icon would correspond to the the local weather conditions. The concept behind this feature is that since the Port serves diners outdoors, it would be useful to get an idea about the current weather through the website.

A second step was to find a typeface to use as a body text, that would pair with the Lobster & Bebas Neue fonts that were already in use. After experimenting with different approaches, we settled on Avenir: it's wide base and shorter x-height provided a strong contrast against Bebas Neue.

Hero Illustrations

The night and day banner illustrations were designed with the aim of using a similar flat design style that was used for the logo design and original icons.

Materialize Framework

Finally, we decided to use the materialize bootstrap theme in development, and this also influenced the bright color palette selection and use of card elements to showcase UI content.
